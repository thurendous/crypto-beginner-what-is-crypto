# 暗号通貨とは何か？ (What is Cryptocurrency?)

初心者向けの「暗号通貨（クリプト）」に関するプレゼンテーション・スライドデッキです。

投資として「どのコインを買えばいいのか」といった価格の話ではなく、歴史的な文脈から**「そもそもお金とは何か？」「なぜ暗号通貨が存在するのか」「どのような技術的・哲学的背景があるのか」**といった「本質」に焦点を当てています。

## ✨ 特徴 (Features)

- **依存関係ゼロ:** ピュアな HTML / CSS / Vanilla JavaScript のみで構成されており、複雑なビルドツールは必要ありません。
- **ダークモード・デザイン:** Web3や暗号通貨のテーマに合わせた、モダンでミニマルなサイバーパンク風デザイン。
- **直感的な操作:** 
  - 画面下部のボタンでのナビゲーション
  - キーボード操作対応（左右の矢印キー、スペースキー）
  - モバイルでのスワイプ操作対応
  - URLのハッシュ（`#1`など）を使った特定のスライドへの直接アクセスルーティング
- **オリジナルFavicon:** スライドのテーマ（ブロック・ノードのつながり）を表現した独自のSVGアイコンを搭載。

## 📖 スライドの主な内容 (Contents)

- **お金の歴史と本質:** 物々交換から金本位制、法定通貨の誕生とその後の変遷。
- **法定通貨の限界:** ニクソン・ショック、インフレーション、カンティヨン効果などの中央集権システムが抱える歴史的課題。
- **サイバーパンク運動と背景:** プライバシーと分散化を求める暗号学者たちの歴史。
- **Bitcoin の誕生と革新:** サトシ・ナカモト、デジタル希少性の発明、2100万枚の上限、半減期の仕組み。
- **Bitcoin vs Ethereum:** 
  - **Bitcoin:** 完全な「価値の保存」とデジタルゴールド。PoW。
  - **Ethereum:** スマートコントラクト、The Merge、EIP-1559（バーン機構）、DeFiネットワークの基盤。
- **DeFi と NFT:** トラストレスな分散型金融、真正なデジタル所有権。

## 🚀 使い方 (How to Run)

このプロジェクトは静的なWebファイルのみで構成されているため、特別なインストールは不要です。

```bash
# ローカルサーバーで起動する場合（Pythonがインストールされている場合）
python3 -m http.server 8000
```
起動後、ブラウザで `http://localhost:8000/` にアクセスしてください。

> 💡 **Tip:** Visual Studio Code をお使いの場合は、「Live Server」拡張機能を使用すると1クリックでローカルプレビューを開くことができます。直接 `index.html` をブラウザにドラッグ＆ドロップしても動作します。

## ☁️ デプロイ (Deployment - Cloudflare Pages)

このプロジェクトは静的ファイルのみで構成されているため、[Cloudflare Pages](https://pages.cloudflare.com/) などのホスティングサービスに非常に簡単に無料でデプロイできます。

Wrangler CLI（Cloudflare のコマンドラインツール）を使用する場合、以下のコマンドを実行するだけです。

```bash
# 現在のフォルダ( . )をデプロイ
npx wrangler pages deploy . --project-name crypto-beginner-what-is-crypto
```

> **注意:** 初回実行時は、ブラウザが開いて Cloudflare アカウントへのログイン・連携が求められる場合があります。
